Encryptron is a simple web application that allows you to encrypt and decrypt files using a password.

How it works:

  1. Upload a file to encrypt.
  2. Enter a password to encrypt the file.
  3. Download the encrypted file.
  4. Upload the encrypted file.
  5. Enter the password to decrypt the file.
  6. Download the decrypted file.
